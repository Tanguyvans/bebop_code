Nous avons deux dossiers. Un dossier contient les codes avec des commentaires et l'autre contient les codes sans commentaire. 
On conseille d'utiliser les codes sans commentaire pour ne pas avoir d'erreur lors du lancement. 

Dans codes nous avons: 
-bebop_inter.py : lancement de l'interface. celui-ci appelle les codes bebop_arrow.py, bebop_control.py et bebop_tracking.py

Si l'on souhaite lancer les codes sans avoir l'interface, nous avons:
-bebop_yolo.py : code pour le tracking de personnes uniquement 
-bebop_fleche.py : code pour la détection de flèches 
-bebop_motion.py : reprend l'ensemble des commandes de déplacement du drone
